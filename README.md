# ESG_-_Financial_Performance_Analysis_Indian_Public_Sector_Banks-
Analyzed relationship between CRISIL ESG scores and financial metrics (ROA, ROE) for 10 major public-sector banks. Quantified impact of ESG factors: top-rated banks achieved ~2× higher ROA and +13.6% ROE compared to others. Presented findings highlighting ESG as a strategic lever for improved financial outcomes.
